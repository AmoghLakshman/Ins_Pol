# Insurance Claims Streamlit Dashboard v2

Files in this package (root of repository):
- app.py
- insurance_data.csv  <-- your uploaded CSV (renamed for consistency)
- README.md
- requirements.txt

Deploy to Streamlit Cloud:
1. Push these files to a GitHub repository (root).
2. In Streamlit Cloud, create a new app and point it to app.py.
3. Streamlit Cloud will install dependencies from requirements.txt and run the app.
